## COMANDOS USADOS DE GIT

1. git init: Inicializa un repositorio local en el directorio actual
2. git checkout rama: Cambia a la rama indicada, con la opción -b puedes crearla si no existe 
3. git pull: Te sincroniza con el repo remoto actualizandote localmente todos los cambios
4. git add : Agrega los archivos indicados a la estpa de staging antes de hacer commit
5. git commit: Confirma los cambios en el repositorio 
6. git push: Sube los cambios desde tu repo local hacia el repo remoto con que esta indicado en el remote origin
7. git remote add origin: Puedes indicar el repo remoto de git en la nube con el que desees sincronizarte.
